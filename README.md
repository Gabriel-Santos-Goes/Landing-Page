Este repositório contém o código-fonte do meu site pessoal. O objetivo é apresentar minha trajetória como desenvolvedor, meus projetos técnicos e facilitar o contato para oportunidades profissionais.

---

## 💻 Sobre o Projeto

O site foi desenvolvido com foco em **Clean Code** e **UX (User Experience)**. Utilizei uma paleta de cores moderna (Índigo e Slate) e tipografia legível para transmitir profissionalismo.

### Principais Funcionalidades:
- **Design Responsivo:** Adaptado para Mobile, Tablet e Desktop.
- **Scroll Reveal:** Animações dinâmicas que surgem conforme a navegação.
- **Sticky Navbar:** Menu fixo com efeito de desfoque (Blur).
- **Smooth Scroll:** Navegação suave entre as seções da página.

---

## 🛠️ Tecnologias Utilizadas

Para este projeto, optei por utilizar tecnologias nativas para demonstrar domínio sólido dos fundamentos:

- **HTML5:** Estruturação semântica.
- **CSS3:** Flexbox, Grid Layout, Variáveis e Media Queries.
- **JavaScript (ES6+):** Manipulação de DOM e eventos de scroll.
- **Google Fonts:** Tipografia 'Inter' para maior legibilidade.
